## 🚀 部署

### GitHub Pages 部署

1. 创建新的 GitHub 仓库
2. 推送代码：
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/li-peirun-website.git
   git branch -M main
   git push -u origin main
   ```
3. 在仓库设置中启用 GitHub Pages (Settings → Pages → Source: main branch)
4. 访问 `https://YOUR_USERNAME.github.io/li-peirun-website/`

### 本地预览

直接用浏览器打开 `index.html` 即可预览。

## 📝 自定义

- 修改头像：编辑 `index.html` 中的 `<img>` 标签 `src` 属性
- 修改文案：直接编辑 HTML 中的文字内容
- 修改配色：编辑 CSS 中的 `:root` 变量
